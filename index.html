Typographic element

They tell the browser to display text in a different way. examples include: mark, em, sub, sup, etc.

sub and sup stand for "subscript" and "superscript". Originally, HTML started out as a tool for scientific and academic work; and these are really useful for science and math, like writing x2+3x+4 = 1 or chemical formulas like H2O.

Nesting

element in html can go inside another element by following some certain rules. the top level element are usually the container for other element. this is an example of nesting a mark element inside a paragraph

Block and Inline element

elements like headings, paragraph are called block element because they create an invisible box (with height, width, and margin) around its content. other examples include: list, video, table, and div

element like em, br, strong, mark, anchor, img, span, that don't create a block are called inline element.

List item

the li element is used to create a list item and it must be contained within an ol or ul opening and closing tags. to create an ordered list item we use ol while ul is used to create an unordered list item.
the closing tag of li is optional same as the p element. when the closing tags are not used for this two element the browser knows where their content stop when it sees another opening tag since this two element can not contain another element.

Adding images

the img element is used to add images to web pages. it consists of two attributes which are src and alt attributes. the src takes the a value which is the image source while the alt attribute is used for image description. the img element is an inline and void element (no content and no closing tag).

In order for a link to refer to a different web site, it has to be an absolute URL. Relative url are links without the http or https.

Sometimes when you're working on a Web page, you don't know what images you're going to use, but you still want a picture to use as a place holder.
Websites like placebear.com and placekitten.com are designed to make this super easy. All you have to do to get an image of any given size is put the width and height of the image at the end of the URL, as in https://placebear.com/200/300.
